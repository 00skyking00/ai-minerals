# VOICE-AND-STYLE.md

This file is the voice/style rule for any prose produced in this repo that ends up on the public portfolio site at `https://johnsondevco.com/ai-minerals/` or in any cover-letter render.

Applies to: `index.qmd`, chapter pages (`bear_cub.qmd`, `regional.qmd`, etc.), and any other reader-facing markdown/Quarto source. **Does not apply** to internal scaffolding (Python comments, JOBSEARCH-BRIEFING.md, HANDOFF entries, configs).

## The filter

Sky shares this work with recruiters at KoBold, Explore Tech, Nvidia, and similar. AI-paper register undermines his case — he is positioning as a programmer with real domain interest and family stakes; if the prose sounds AI-authored, the case is dead.

**Retire these AI tells:**

- "Each X asks the same question…" / "X asks the question of…"
- "What separates X from Y is…" / "What makes X different is…"
- "Underneath each X sits a Y" / "X sits beneath Y"
- "intellectual move," "mathematical machineries," "essential pivot," "epistemic stance," "the same posture either way"
- Perfect-parallel rhythms in adjacent sentences
- Clause pile-ups with multiple "on a [X] or [Y]" inserts
- Methodology paragraphs that read like an abstract
- Third-person voice on first-person work — refusing to say "my claim" when it is his claim
- Crisp enumerated lists of academic concepts where prose would fit
- Tidy thesis-restating closers

**Replace with:**

- First person — name the people, places, dollar amounts, dates
- Varied sentence length — let some run long, let some be three words
- Programmer-humility where Sky is operating outside his main field: "I'm a programmer, not a geologist, but…"
- Admit what's hard, what didn't work, what he's unsure about
- Allow contractions and casual transitions
- Light dry closers that puncture formality

## Reference voice

The canonical in-voice anchor is `~/src/jobsearch/cover-letters/cover-letter-source.qmd` — Sky's KoBold / Explore Tech cover letter. Any prose page on this site should be comparable in register.

## Test

Before committing any rewrite or new prose: read it aloud. If a sentence wouldn't come out of Sky's mouth in conversation, rewrite it.
